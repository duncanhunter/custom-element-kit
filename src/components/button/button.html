Attributes

Attribute Description
type Sets the button type. Possible values are button, submit, and reset. Defaults to button.
disabled Disables the button when present.
loading Shows a loading indicator when present.
arrow Displays an arrow icon when present.
variant Sets the styling variant of the button. Possible values are primary, ascent, outline, text, and link.
size Sets the size of the button. Possible values are small, medium, and large.
icon-only If present, styles the button for icon-only display without text.
button-aria-label Sets an ARIA label for accessibility.

Slots

Slot Name Description
(default) The text content of the button.
start Content to display before the button's text (e.g., icons).
end Content to display after the button's text (e.g., icons).

CSS Parts

Part Name Description
button The native button element.
label The label element inside the button.
loading-icon The loading spinner displayed when loading is present.
arrow-icon The arrow icon displayed when arrow is present.

Events

Event Description
click Dispatched when the button is activated.

Methods

Method Description
focus() Moves keyboard focus to the button.
