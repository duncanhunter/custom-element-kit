Code Block (cek-code-block)


package main import "fmt" func main() { fmt.Println("Hello, World!") }

Limitations

  • If the code block contains live code like other custom elements it needs to be wrapped in a <template></template> to avoid the custom elements being defined and rendered and code executed.
