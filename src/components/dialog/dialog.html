Open Attribute

The open attribute can be used to open the dialog in non-modal mode.

Modal Attribute

The open modal attributes can be used together to open the dialog in modal mode.

Attributes

Attribute Description
open Displays the dialog in non-modal mode when present.
modal If present, the dialog is shown as a modal dialog.
open-modal Displays the dialog in modal mode when present.

Slots

Slot Name Description
header Content to display in the header of the dialog.
header-actions Content to display in the header actions area.
default The main content of the dialog.
footer Content to display in the footer of the dialog.

CSS Parts

Part Name Description
dialog The native dialog element.
header The header element of the dialog.
content The content element of the dialog.
footer The footer element of the dialog.

Events

Event Description
close Dispatched when the dialog is closed.

Methods

Method Description
show() Shows the dialog as a non-modal dialog.
showModal() Shows the dialog as a modal dialog.
close(result) Closes the dialog with an optional result.
