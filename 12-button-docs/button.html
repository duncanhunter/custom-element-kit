Button (cek-button)

Variants

The button component supports several variants for different styles and use cases. Use the variant attribute to change the visual style of the button.

Sizes

Small, medium (default), and large sizes are available. Use the size attribute to adjust the size of the button.

Anchor Functionality

The button component can also function as a link. Use the href attribute to transform the button into an anchor element. Optionally, you can use the target attribute to specify where to open the linked document.

Attributes

Attribute Description
type Sets the button type. Possible values are button, submit, and reset. Defaults to button.
disabled Disables the button when present.
loading Shows a loading indicator when present.
arrow Displays an arrow icon when present.
variant Sets the styling variant of the button. Possible values are primary, ascent, outline, text, and link.
size Sets the size of the button. Possible values are small, medium, and large.
icon-only If present, styles the button for icon-only display without text.
button-aria-label Sets an ARIA label for accessibility.
href Transforms the button into an anchor element with this href.
target Sets the target attribute of the anchor when href is present. Defaults to _self.

Slots

Slot Name Description
(default) The text content of the button.
start Content to display before the button's text (e.g., icons).
end Content to display after the button's text (e.g., icons).

CSS Parts

Part Name Description
button The native button or anchor element.
label The label element inside the button.
loading-icon The loading spinner displayed when loading is present.
arrow-icon The arrow icon displayed when arrow is present.

Events

Event Description
click Dispatched when the button is activated. For a link, dispatched upon navigation activation.

Methods

Method Description
focus() Moves keyboard focus to the button or anchor.
