Icon (cek-icon)

Basic Usage

The icon component is used to display SVG icons. Use the name attribute to specify the icon to display.

Sizes

Use the size attribute to adjust the size of the icon.

Colors

Use CSS to style the color of the icon.

Attributes

Attribute Description
name Specifies the name of the icon to display.
size Specifies the size of the icon. Accepts any valid CSS size value (e.g., 1em, 24px).

CSS Parts

Part Name Description
icon The SVG element representing the icon.
