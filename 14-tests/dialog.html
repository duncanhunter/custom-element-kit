Accessibility

When implementing a dialog, it is important to set the initial focus appropriately. Use the autofocus attribute to ensure the initial focus is on the most important element. If unsure, set the focus on the <dialog> element itself.

Provide a mechanism for users to close the dialog, such as a close button. Modal dialogs opened with showModal() can be dismissed with the Esc key, which should be maintained for accessibility. If you want to prevent the dialog from closing with the Esc key, use the no-esc-close attribute.

The native <dialog> element provides built-in accessibility features. When using showModal(), the dialog is treated as modal with aria-modal="true". For non-modal dialogs, aria-modal="false" is used. Ensure everything outside the modal dialog is inert using the inert attribute.

For critical dialogs where the user must make a choice, you can use role="alertdialog". Additionally, you can choose not to add a close icon and not use the close-on-outside-click attribute and prevent close on escape key with no-esc-close attribute to ensure the user makes a choice within the dialog.

Attributes

Attribute Description
open Displays the dialog in non-modal mode when present.
modal If present, the dialog is shown as a modal dialog.
size The size of the dialog. Can be "small", "medium", or "large".
close-on-outside-click If present, the dialog will close when the backdrop or outside of the dialog is clicked.
no-esc-close If present, the dialog will not close when the ESC key is pressed.

Slots

Slot Name Description
header Content to display in the header of the dialog.
default The main content of the dialog.
footer Content to display in the footer of the dialog.

CSS Parts

Part Name Description
dialog The native dialog element.
header The header element of the dialog.
content The content element of the dialog.
footer The footer element of the dialog.

Events

Event Description
close Dispatched when the dialog is closed.

Methods

Method Description
show() Shows the dialog as a non-modal dialog.
showModal() Shows the dialog as a modal dialog.
close(result) Closes the dialog with an optional result.
